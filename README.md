# AI Resume Optimizer 🚀

An intelligent tool designed to bridge the gap between job seekers and Applicant Tracking Systems (ATS). This project uses Large Language Models to analyze resumes against specific job descriptions.

## ✨ Features
* **ATS Scoring:** Get a percentage match of how well your resume fits the job.
* **Keyword Analysis:** Identifies missing critical keywords from the JD.
* **Formatting Suggestions:** Tips on improving resume structure for better readability.

## 🛠️ Tech Stack
* **Language:** Python
* **LLM:** Google Gemini Pro API
* **Environment:** Google Colab / Local Python
